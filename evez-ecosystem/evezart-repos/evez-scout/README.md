# EVEZ Scout — Autonomous Research Agent

Self-directed research and intelligence gathering for the EVEZ stack.

## Quick Start

```bash
git clone https://github.com/EvezArt/evez-scout.git
cd evez-scout
pip install -r requirements.txt
export GROQ_API_KEY=***
python scout.py
```

## What It Does
- Autonomous web research via SearXNG
- Document analysis and summarization
- Intelligence reports generated via Groq LLM
- MAES event integration for distributed awareness

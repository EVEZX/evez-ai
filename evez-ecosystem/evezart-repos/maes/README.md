# MAES — Multi-Agent Event Spine

The central event bus for the EVEZ ecosystem. All 8 autonomous agents (SPINE, TRUNK, DEPLOY, VAULT, HARVEST, SCOUT, WITNESS, CAIN) communicate through MAES.

## What It Does
- Central event routing for all EVEZ agents
- Asynchronous message passing
- Event logging and replay
- Health monitoring of all connected agents

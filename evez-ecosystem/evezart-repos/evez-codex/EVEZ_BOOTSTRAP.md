# EVEZ Bootstrap

Mission:
Build EVEZ as a modular OpenClaw agent system.

Modules:
- EVEZ-Core: orchestration
- EVEZ-Logic: reasoning
- EVEZ-Temporal: sequence analysis
- EVEZ-Vector: memory and retrieval
- EVEZ-Code: tooling and code execution

Startup:
1. Install Ollama
2. Pull qwen3:8b
3. Install ChromaDB
4. Connect OpenClaw

Runtime flow:
input -> router -> module -> memory -> output

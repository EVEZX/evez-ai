8 Risk Factors

Each factor is normalized 0–1, weighted, and mapped to the 300–850 score range. Adverse action codes (AA01–AA08) are auto-generated for factors below threshold.

Payment History

35% The single largest factor. On-time payments, delinquency patterns.

Credit Utilization

20% Balance-to-limit ratio on revolving accounts.

Credit Age

15% Length of credit history in years. Longer = better.

Credit Mix

10% Variety of account types (revolving, installment, mortgage).

New Inquiries

5% Recent hard inquiries. More = riskier.

Debt-to-Income

5% Monthly debt payments vs. income.

Derogatory Marks

5% Collections, bankruptcies, tax liens.

Total Accounts

5% Number of established credit accounts.

API Reference

RESTful, JSON in/out, Bearer token auth.

POST /score

Score a single applicant. Returns score, grade, decision, adverse actions, full factor breakdown, and compliance data.

POST /batch

Score up to 100 applicants in one request. Pass applicants array or generate sample data.

GET /model-info

Model version, factor weights, grade thresholds, compliance standards.

GET /usage

Monthly request count, credits remaining, daily breakdown by endpoint.

POST /billing/checkout

Stripe Checkout session for plan upgrade. Handles subscription lifecycle.

POST /billing/webhook

Stripe webhook handler. Auto-upgrades API keys on payment completion.

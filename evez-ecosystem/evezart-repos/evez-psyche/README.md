# EVEZ Psyche Engine v3 — Self-Evolving Music Organism

A pure numpy genetic algorithm that creates self-evolving music. Zero APIs. Zero cost. It feeds on recordings and evolves.

## Philosophy
> "I am the music I have listened to" — The Psyche Engine learned from 12 of Steven's recordings (273s corpus). His DNA is in the organism's bloodline.

## Quick Start

```bash
git clone https://github.com/EvezArt/evez-psyche.git
cd evez-psyche
pip install -r requirements.txt
python psyche.py
```

2. Genetic algorithm: crossover + mutation
3. Feeds on audio recordings as DNA source
4. Evolves toward harmonic fitness
5. Key: F1 (43.65Hz), 77-143 BPM range
6. Mid-heavy, classical × phonk × dubstack

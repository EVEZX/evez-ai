# EVEZ Pulse — System Health Monitor

Real-time health monitoring and alerting for EVEZ microservices.

## What It Does
- Health checks for all 25+ EVEZ services
- Alert routing via Slack and Telegram
- Uptime tracking and SLA reporting
- Auto-restart on failure detection

## Quick Start

```bash
git clone https://github.com EVEZArt/evez-pulse.git
cd evez-pulse
pip install -r requirements.txt
python pulse.py
```
